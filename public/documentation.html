Documentation for Movie API

The objective of this project is to build a server-side component of a movies web application. This will take the form of a REST API. Through the functionality afforded by the API, users will be able to search for information on different movies, genres or directors. They will be able to sign up, update their personal information and save their favourite movies.

Endpoints and Request Methods

The resources a user could find on this application can be accessed using the URL endpoints and request methods below:

Business Logic URL HTTP Method Request body data format Response body data format
Return a list of all movies /movies GET None JSON object holding data about the available movies
Return data about a movie by title /movies/:Title GET None JSON object holding data about a movie
Return data about a genre by name /movies/genre/:genreName GET None JSON object holding data about a genre
Return data about a director /movies/directors/:directorName GET None JSON object holding data about a director
Create a new user profile /users POST JSON, e.g.

{"Username": "Saeed",
