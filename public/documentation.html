Documentation for Movie API

The objective of this project is to build a server-side component of a movies web application. This will take the form of a REST API. Through the functionality afforded by the API, users will be able to search for information on different movies, genres or directors. They will be able to sign up, update their personal information and save their favourite movies.

Endpoints and Request Methods

The resources a user could find on this application can be accessed using the URL endpoints and request methods below:

Business Logic URL HTTP Method Request body data format Response body data format
Return a list of all movies /movies GET None JSON
Return data about a single movie by title /movies/:title GET None JSON
Return data about a genre by name /movies/genre/:genreName GET None JSON
Return data about a director /movies/directors/:directorName GET None JSON
Allow new user to register /users POST JSON JSON
Allow users to update their user info, e.g. name /users/:id PUT JSON JSON
Allow users to add a movie to their list of favorites /users/:id/:movieTitle POST Text message confirming the movie has been successfully added
Allow users to remove a movie from their list of favorites /users/:id/:movieTitle DELETE None Text message confirming the movie has been deleted
Allows existing users to deregister /users/:id/ DELETE None Text message confirming the user has been deleted
